# backend-sho-mi-ke-ko

## Requisitos
- Docker
- Python

## Tecnologias
- [Orator] (https://orator-orm.com/docs/0.9)

## Setup

### Rodar com docker

1. Rodar o projeto
```
  make up
```
2. A API está rodando na porta 5000
  - Para testar, envie uma requisição para http://localhost:5000/

### Rodar na máquina

1. Rodar o projeto
```
  make up-machine
```
2. A API está rodando na porta 5000
  - Para testar, envie uma requisição para http://localhost:5000/
